Why does this exist?

After Cimoooooooo's progression series launched, my brother and I decided to try it ourselves. We like using automated simulators like EDOPro rather than Dueling Book.

However, we found it difficult to manage our collections after opening several sets because we'd have just so much. I created this tool so we would know exactly how much of each card we could use when deck building.

How can I help development?

While I feel like this is feature complete right now, if you want to add something or run into a bug, join me on GitHub.
